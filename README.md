# Usage
<!-- usage -->
```sh-session
$ npm install -g ignite
$ ignite COMMAND

$ ignite (-v|--version|version)
ignite/1.0.0 linux-x64 node-v10.19.0
$ ignite --help [COMMAND]
USAGE
  $ ignite COMMAND
...
```
